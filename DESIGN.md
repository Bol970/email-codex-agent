# Design Context

Use a restrained Notion-like interface: dense but calm, mostly white surfaces, hairline borders, compact controls, clear active states, and no marketing-style hero sections.

Use shadcn/ui composition patterns, lucide icons for tool buttons, 8px or smaller radii, semantic color tokens, and predictable split-pane navigation.

The first screen is the actual product: inbox sidebar, thread list, reading pane, and Codex panel. Avoid decorative backgrounds, floating section cards, and oversized headings.
