# Email Codex Agent

Local-first email workspace for one AgentMail inbox. The product combines a quiet mail reader, manual reply controls, draft-first agent automation, and an embedded Codex app-server session backed by the user's existing ChatGPT subscription.

Primary user: one technical owner reading and triaging inbound messages locally.

Core promise: make email easier to scan, summarize, label, and draft responses to, without letting an agent send mail without explicit approval.
